PathingLib Documentation

Pathing DLL

Allows creating graphs and finding paths by using Dijkstra or ALT algorithms.

Example Usage

Dijkstra

PathingLib::Graph g = PathingLib::Graph::loadGraphFromFile("graphNodes.txt", "graphEdges.txt"); PathingLib::Path p = PathingLib::Dijkstra::getPath(0, 5, g); cout << "Path distance: " << p.getDistance('m');

ALT

PathingLib::Graph g = PathingLib::Graph::loadGraphFromFile("graphNodes.txt", "graphEdges.txt"); PathingLib::ALT alt(g, 4); PathingLib::Path p = alt.getPath(0, 1000); cout << "Path distance: " << p.getDistance('m');
